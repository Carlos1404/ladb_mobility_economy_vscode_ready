# Mobility and Economy Analysis in Latin American Cities

This project explores the relationship between urban mobility indicators and economic variables in Latin American cities during 2024.

## Why this version exists

The original course notebook expected two raw files from a managed learning environment:

- `tomtom_traffic.csv`
- `oecd_city_economy.csv`

Those files were loaded from `/datasets/`, which usually only exists inside the original platform. Because the raw files are not available locally, this repository is configured to run in VS Code using the clean dataset generated by the project:

```text
data/processed/ladb_mobility_economy_2024_clean.csv
```

This file is not treated as raw data. It is the processed dataset used to reproduce the final analysis and visualizations.

## How to run in VS Code

1. Open this folder in VS Code.
2. Create a virtual environment:

```bash
python -m venv .venv
```

3. Activate the environment.

On macOS/Linux:

```bash
source .venv/bin/activate
```

On Windows PowerShell:

```powershell
.venv\Scripts\Activate.ps1
```

4. Install dependencies:

```bash
pip install -r requirements.txt
```

5. Open the notebook:

```text
notebooks/ladb_mobility_economy_analysis_vscode.ipynb
```

6. Select the `.venv` Python kernel and run all cells.

## Alternative: run as a Python script

You can also run the analysis from the terminal:

```bash
python src/run_analysis.py
```

This generates charts in the `outputs/` folder.

## Main dataset

The analysis uses 2024 city-level data with mobility and economy indicators, including:

- Traffic delay
- Live traffic index
- Jam length and jam count
- Travel time indicators
- GDP per capita
- Unemployment rate
- PM2.5
- Population

## Notes

This is an exploratory analysis. The results show associations between mobility and economic indicators, not causal effects.
